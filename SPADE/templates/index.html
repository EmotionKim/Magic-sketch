Guide

사용 설명서


Magic Sketch는 캔버스에서 그림을 그린 이미지를 GauGAN을 이용하여 매우 사실적으로 보여주는 디자인 도구입니다.


그림 그리기
1. 브러쉬 버튼 누르고, 브러쉬 크기 선택하고, 원하는 물체 클릭하면,
2. 그림 그릴 준비 끝!
3. 캔버스 위에서 마우스로 그림을 그려보세요.

이미지로 전환하기
그림을 다 그렸나요? 그럼 가운데 토끼를 눌러보세요!
오른쪽 캔버스에 이미지가 뜬답니다.

버튼 설명서
캔버스 상단 | (왼쪽부터 순서대로) 브러쉬 크기, 펜 / 지우개 / 채우기, 뒤로가기 / 되돌리기 / 캔버스 초기화, 다운로드
캔버스 하단 | 브러쉬 색상. 각 색상에 해당하는 물체가 사실적으로 나타납니다.
매직스타트(변환) | 캔버스(좌측) 이미지를 사실적으로 변환합니다. 결과는 우측에 나타납니다.
